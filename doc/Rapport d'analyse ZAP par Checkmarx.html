          Description de l'alerte

          La politique de sécurité du contenu (CSP) constitue une couche de sécurité supplémentaire permettant de détecter et d'atténuer certains types d'attaques, notamment les attaques de type Cross-Site Scripting (XSS) et par injection de données. Ces attaques servent à diverses fins, allant du vol de données à la défiguration de sites web, en passant par la distribution de logiciels malveillants. La CSP fournit un ensemble d'en-têtes HTTP standard permettant aux propriétaires de sites web de déclarer les sources de contenu autorisées que les navigateurs peuvent charger sur la page. Parmi ces sources figurent JavaScript, CSS, les cadres HTML, les polices, les images et les objets intégrables tels que les applets Java, ActiveX, ainsi que les fichiers audio et vidéo.
